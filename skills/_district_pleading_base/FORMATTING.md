# FORMATTING.md — District Court formatting specification

Plain-language formatting reference for `district-court-drafting`. The `state-config.md` in the user's case folder supplies State-specific overrides; this file is the pan-India baseline.

---

## Paper & layout

- **Paper size:** A4 (most States · default)
  - Some State Civil Manual practices specify foolscap or Legal — verify against your state-config
- **Print:** one-side preferred · two-side where State Civil Manual expressly permits

## Typography

- **Font family:** Times New Roman (default)
  - Many State Civil Manuals permit Arial / Calibri / Cambria — verify Registry acceptance
- **Font size body:** 14 point (default)
  - 12 point acceptable per some State Manuals (e.g., Maharashtra Civil Manual at District level)
- **Font size headings:** 14 point bold
- **Line spacing:**
  - **1.5** — standard for most States
  - Some Original Side practices (urban-court traditions) prefer double-spacing — verify state-config

## Margins

- **Left (binding side):** 4 cm
- **Top / bottom / right:** 2.5 cm

## Page numbers

- Centred at bottom
- Continuous pagination — main pleading through Schedule of Property through List of Documents through accompanying applications

## Section headers

- Title Case, bold, 14 point (most States)
- Some State practices (e.g., Bombay Civil Manual) prefer spaced section heads — per state-config

## Cause Title block

- Court header: `IN THE COURT OF THE [JUDICIAL OFFICER DESIGNATION per state-config.md Section 2] AT [Place], DISTRICT [District Name]`
- Case number line: `[CASE TYPE] No. _____ of [Year]`
  - Some States use specific case-type prefixes (e.g., "Regular Civil Suit", "Special Civil Suit", "R.C.S.", "S.C.S.", "C.C.", "Complaint Case") — per state-config
- Parties block: name + age + occupation + address

## Exhibit / Annexure marker

- Inline marker — per state-config Section 8 (`exhibit_prefix`)
  - Common pan-India default: `EXHIBIT-A, B, C, ...`
  - Some State practices use `Annexure-A` or `Doc. No. 1`
- Consolidated `LIST OF DOCUMENTS / EXHIBITS` table at end of pleading

## Mandatory blocks for civil pleadings

1. **Cause of Action** — distinct numbered paragraph (Order VII Rule 1(e) CPC) — mandatory in every plaint
2. **Jurisdiction** — territorial + pecuniary + subject-matter (Order VII Rule 1(f), (g))
3. **Limitation** — specific Article of Schedule to the Limitation Act 1963
4. **Court-Fee** — value of suit + State Court-Fees Act citation per state-config Section 4
5. **Schedule of Property** — mandatory for immovable-property suits (boundaries N/S/E/W + survey + area + mutation entries)
6. **Verification** — CPC Order VI Rule 15 verbatim
7. **Place + Date + Advocate signature** — with Bar Council enrolment number

## Mandatory blocks for criminal pleadings (BNSS-governed)

1. **Date and place of occurrence** — NOT "cause of action" (criminal terminology)
2. **FIR / Complaint particulars** (where applicable) — FIR number + date + police station + sections invoked
3. **Custody status of accused** (for bail / anticipatory bail / 313 statement)
4. **Dual citation** — every reference to CrPC paired with BNSS, every IPC paired with BNS, every IEA paired with BSA
5. **Verification per CPC Order VI Rule 15** (or per BNSS Section 297 for criminal-specific affidavits)
6. **Place + Date + Advocate signature**

## Vakalatnama

- Separate document
- Per State Bar Council standard format · stamped per State Stamp Schedule
- See state-config Section 7

## Accompanying applications (case-type-specific)

- Each application repeats own Cause Title + own facts + own prayer + own Verification / Affidavit + own Advocate signature

## State-specific exceptions to verify in your state-config

- **Tamil Nadu:** "Subordinate Judge" / "District Munsiff" terminology; Madras Civil Manual; Tamil-language pleadings permitted in some courts
- **Kerala:** "Sub Court" / "Munsiff Court" terminology; foolscap paper conventional; book-stitched (verify Registry)
- **Madhya Pradesh / Chhattisgarh:** "Civil Judge Class I / Class II" terminology
- **Bihar / Jharkhand:** "Sub Judge" / "Munsif" (older terminology)
- **AP / Telangana:** "Senior Civil Judge" / "Junior Civil Judge"
- **Punjab / Haryana / Chandigarh:** "Senior Sub Judge" / "Sub Judge"; Punjab Courts Act 1918
- **Delhi (NCT):** "Senior Civil Judge" / "Civil Judge"; "Chief Metropolitan Magistrate" / "Metropolitan Magistrate" terminology for criminal courts
- **Maharashtra / Gujarat / West Bengal / Odisha / UP / Uttarakhand:** "Civil Judge Senior Division / Junior Division" (or parenthetical form in Odisha)
