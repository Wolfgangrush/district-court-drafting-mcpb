## Section 2 — Plaint: Cause of Action paragraph (mandatory per Order VII Rule 1(e))

```
CAUSE OF ACTION

The cause of action for the present suit arose on <DD.MM.YYYY> at [Place] when [the specific event giving rise to the suit — breach / refusal / denial / wrongful act]. The cause of action is a continuing one in that [where applicable]. The Defendant has thereby [denied / breached / refused] the Plaintiff's [right / interest], rendering the present suit necessary.
```

## Section 3 — Written Statement: paragraph-wise traversal opener

```
The Defendant above-named submits the present Written Statement in answer to the Plaintiff's plaint:

PRELIMINARY OBJECTIONS:

1. [Maintainability / limitation / jurisdiction objections, where any.]

REPLY ON MERITS:

With reference to paragraph 1 of the plaint, the contents thereof are admitted to the extent that [admitted facts]; the contents are denied to the extent that [denied facts].

With reference to paragraph 2 of the plaint, the contents thereof are denied as false and unfounded. The true position is that [Defendant's version of facts].

[Continue paragraph-wise — every plaint paragraph addressed by number. Absence of specific denial is treated as admission under Order VIII Rule 5 CPC.]
```
